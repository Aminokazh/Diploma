Dataset is available at: https://universe.roboflow.com/ayesha-ovgn4/abnormal-behaviour

The dataset is not included in this repository due to size limitations.

To run the application correctly, place the dataset in the following directory:
abnormal_behaviour/

This project uses Streamlit as the user interface.

To start the application, run the following command in the project root directory:

python -m streamlit run app.py
